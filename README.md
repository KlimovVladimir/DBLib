### Building:
```
make
```

### Installing:
```
sudo make install
LD_LIBRARY_PATH=/usr/local/lib/
export LD_LIBRARY_PATH
```

### Running:
```
dbutil <pathToFileData1> <pathToFileData2> <pathToFileOut>
```
